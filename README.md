# 🕹️ Retro Arcade Collection

Willkommen in der **Retro Arcade Collection**! Eine handgefertigte Sammlung klassischer Arcade-Spiele, die den Charme der 80er Jahre direkt in deinen Browser bringt.

Alles läuft auf **HTML5 Canvas** und reinem **JavaScript** – keine Game-Engines, nur purer Code und Vektor-Liebe.

[**👉 JETZT SPIELEN (LIVE DEMO)**](https://amarow.github.io/games/)

---

## 🚀 Die Spiele

### 1. Puzzle (NEU!)

Ein klassisches Legepuzzle im Stil bekannter 1000-Teile-Kartons.

* **Motive:** Feste Auswahl (Schloss, Berglandschaft, Flusslandschaft) oder eigenes Foto per Upload.
* **Teilezahl:** Wählbar zwischen 50, 100, 500 und 1000 Teilen.
* **Gameplay:** Teile aus der Ablage per Drag & Drop (Maus/Finger) ins Zielfeld ziehen; passende Nachbarn rasten mit Animation ein.
* **Extras:** Sortier-Funktion für die Ablage (Rand-/Innenteile), automatische Vorführung, Fortsetzen eines gespeicherten Spielstands.

### 2. Death Star Assault

Ein First-Person-Trench-Run in klassischer Vektorgrafik.

* **Mission:** Fliege durch den engen Graben des Todessterns.
* **Gameplay:** Weiche den Barrieren vertikal aus und schieße TIE-Fighter ab.
* **Finale:** Triff den Lüftungsschacht und entkomme der spektakulären Explosion.
* **Stil:** Grüne Drahtgitter-Optik, 3D-Perspektive, atmosphärischer Schimmer.

### 3. Space Invaders (1980 Edition)

Der Urvater aller Shooter.

* **Mission:** Verteidige die Erde gegen Wellen von Aliens.
* **Feature:** Klassische Pixel-Art, zerstörbare Bunker und das berüchtigte UFO.

### 4. Galaga & Galaga Vector

Zwei Varianten des Kult-Klassikers.

* **Galaga:** Die bekannte Version mit Formationen und Traktorstrahlen.
* **Vector Edition:** Eine spezielle Interpretation im Stil des Vectrex-Konsolensystems.

## 🎮 Steuerung

Das Hauptmenü und die Shooter sind komplett tastaturgesteuert; das Puzzle wird per Maus bzw. Touch gespielt.

| Taste / Eingabe   | Funktion                              |
|:----------------- |:-------------------------------------- |
| **Pfeiltasten**    | Bewegen / Navigieren im Menü           |
| **SPACE**          | Feuern / Bestätigen / Starten          |
| **ENTER**          | Spiel auswählen                        |
| **F**              | Vollbildmodus (Fullscreen)             |
| **ESC**            | Zurück zum Hauptmenü                   |
| **Maus / Finger**  | Puzzle: Teile per Drag & Drop einpassen |

## 🛠️ Technologie

Dieses Projekt wurde erstellt, um die Möglichkeiten von modernem Web-Code für Retro-Gaming zu zeigen.

* **Frontend:** HTML5, CSS3 (Grid-Layout, CRT-Scanline-Effekte, Neon-Glow).
* **Engine:** Custom JavaScript (Canvas API).
* **Grafik:**
  * *Death Star:* Prozedurale 3D-Projektion von Vektoren (Vertices & Edges).
  * *Puzzle:* Clientseitige Zerlegung von Bildern (Canvas) in Teile mit klassischen Zapfen/Buchten.
  * *Icons:* Handgecodete inline SVGs.
* **Entwicklung:** Erstellt mit Unterstützung von Google Gemini (KI-Assistent).
